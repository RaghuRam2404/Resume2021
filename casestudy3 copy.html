Zoho CRM - Zoho Finance Suite Integration
It was the first major project given to me. The integration has to be established between Zoho CRM and Zoho Books, Expense, Invoices. It should enable the 2-way data sync between both the products. Create proper reports and dashboard inside Zoho CRM with the Zoho Finance's data.
"Developed in the span of 5 months.
Self Work contribution"
Product Features

1. Create Modules inside Zoho CRM: For each type of data in Zoho Finance, we'll create a new module in Zoho CRM.
2. Data will be in two way sync between Zoho CRM and Zoho Finance
3. Create default reports and dashboard with Zoho Finance's data, so that users can see it inside Zoho CRM itself
